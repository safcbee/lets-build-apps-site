CODEX GOAL — WEBSITE V5: PREMIUM PRODUCT EXPERIENCE

Mission

Create a completely new V5 website experience for Let’s Build Apps.

This is not a continuation of V4.

This is not a redesign of the existing layout.

This is a premium product-marketing experience built from first principles.

The goal is to create a website that makes visitors want to install the apps.

When someone lands on the homepage they should immediately think:

“These apps look beautiful. I want to try them.”

V4 remains the safety net.

V5 is the ambitious version.

Do not destroy V4.

Build V5 separately first.

⸻

Core Principle

Current V4 answers:

What are these apps?

V5 must answer:

Why should I care?

Everything should be built around:

* visual storytelling
* product desire
* emotion
* simplicity
* clarity
* credibility

Website Philosophy

The website should feel like:

Apple introducing a family of products.

Not:

A repository of projects.

The products should feel connected.

The user should understand:

* what each app does
* why it exists
* how it helps

without reading large blocks of text.

Motion & Interaction

Add tasteful interaction.

Examples:

* subtle hover effects
* card expansion
* screenshot transitions
* scroll-triggered reveals
* parallax effects
* animated section transitions

Avoid:

* gimmicks
* excessive motion
* performance-heavy effects

Performance remains important.

⸻

Screenshots First

Current site still relies too heavily on copy.

V5 should be screenshot-led.

Rule:

Show.
Then explain.

not:

Explain.
Then show.

⸻

Mobile First

The mobile experience must feel premium.

Requirements:

* large touch targets
* beautiful scrolling
* visual storytelling
* strong hierarchy

The mobile site should not feel like a compressed desktop site.

⸻

Success Criteria

A first-time visitor should:

1. Understand the portfolio within seconds.
2. Want to explore the products.
3. Feel the apps are real and polished.
4. Remember the brand afterwards.
5. Be more likely to install an app or join TestFlight.

If someone says:

“That looks like a collection of apps.”

V5 has failed.

If someone says:

“That looks like a product company.”

V5 has succeeded.
